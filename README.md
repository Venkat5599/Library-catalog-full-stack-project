## 🌟 **Overview**

The **Library Book Catalog** is a comprehensive digital library management system that revolutionizes how libraries operate. Built with modern web technologies, it offers an intuitive interface for both librarians and members to manage books, borrowing, and library operations efficiently.

### ✨ **Key Highlights**
- 🎨 **Modern UI/UX** - Clean, responsive design with animated backgrounds
- 📱 **Mobile Responsive** - Works seamlessly on all devices
- 🔐 **Secure Authentication** - JWT-based auth with role-based access
- 📊 **Real-time Analytics** - Interactive charts and statistics
- 🖼️ **Rich Book Covers** - High-quality book cover images
- 🔍 **Advanced Search** - Smart filtering and search capabilities

## 🚀 **Quick Start**

### 📋 **Prerequisites**
- ![Node.js](https://img.shields.io/badge/Node.js-v16+-339933?style=flat-square&logo=node.js) 
- ![MongoDB](https://img.shields.io/badge/MongoDB-Atlas%20or%20Local-47A248?style=flat-square&logo=mongodb)
- ![Git](https://img.shields.io/badge/Git-Latest-F05032?style=flat-square&logo=git)

### ⚡ **Installation**

1. **Clone the repository**
   ```bash
   git clone https://github.com/Venkat5599/Library-catalog-full-stack-project-.git
   cd Library-catalog-full-stack-project-
   ```

2. **Install all dependencies**
   ```bash
   npm run install-all
   ```

3. **Environment Setup**
   ```bash
   cd server
   # The .env file is already configured with MongoDB Atlas
   # No additional setup needed!
   ```

4. **Seed the database**
   ```bash
   npm run seed
   ```

5. **Start the application**
   ```bash
   npm run dev
   ```

6. **Access the application**
   - 🌐 **Frontend**: http://localhost:3000
   - 🔧 **Backend API**: http://localhost:5000

<details>
<summary><strong>🔐 Authentication Endpoints</strong></summary>

| Method | Endpoint | Description | Auth Required |
|--------|----------|-------------|---------------|
| `POST` | `/api/auth/register` | Register new user | ❌ |
| `POST` | `/api/auth/login` | User login | ❌ |
| `GET` | `/api/auth/me` | Get current user | ✅ |
| `PUT` | `/api/auth/profile` | Update profile | ✅ |

</details>

<details>
<summary><strong>📚 Book Management</strong></summary>

| Method | Endpoint | Description | Auth Required |
|--------|----------|-------------|---------------|
| `GET` | `/api/books` | Get all books with filters | ❌ |
| `GET` | `/api/books/:id` | Get single book | ❌ |
| `POST` | `/api/books` | Add new book | ✅ Admin |
| `PUT` | `/api/books/:id` | Update book | ✅ Admin |
| `DELETE` | `/api/books/:id` | Delete book | ✅ Admin |

</details>

| `GET` | `/api/borrows/my-borrows` | Get user borrows | ✅ |

## 🎨 **Design System**

### **Color Palette**
- 🔵 **Primary Blue**: `#1a56db` - Main brand color
- ⚪ **Clean White**: `#ffffff` - Background and cards
- 🔘 **Subtle Gray**: `#64748b` - Text and borders
- 🟢 **Success Green**: `#10b981` - Success states
- 🔴 **Error Red**: `#ef4444` - Error states

### **Typography**
- **Primary Font**: Plus Jakarta Sans (Modern, clean)
- **Display Font**: Sora (Headers and branding)
- **Monospace**: For code and IDs

### **Components**
- 🎭 **Animated Background**: ShapeGrid component with diagonal movement
- 📊 **Interactive Charts**: Chart.js with custom styling
- 🎨 **Modern Cards**: Clean design with subtle shadows
- 🔘 **Smart Buttons**: Hover effects and loading states
